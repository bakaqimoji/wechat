# 健身教练找搭档小程序 - 头像上传指南

## 📸 头像上传位置

### 1. 个人中心页面
**位置**: 底部导航栏点击"我的"

**操作步骤**:
1. 点击头像上的📷相机图标
2. 选择上传类型：
   - **拍照**: 直接拍照上传
   - **从相册选择**: 从手机相册选择图片
   - **选择教练头像**: 为教练上传头像

### 2. 教练头像上传
**适用对象**: 健身教练本人

**操作步骤**:
1. 在个人中心点击头像📷图标
2. 选择"选择教练头像"
3. 从列表中选择自己（如刘教练）
4. 选择"上传头像"或"使用默认"
5. 如果选择上传，会弹出图片选择器

## 🖼️ 支持的图片格式
- **格式**: JPG、PNG、GIF
- **大小**: 建议不超过2MB
- **尺寸**: 自动压缩，显示为正方形

## 💾 头像存储
- 上传的头像保存在本地存储中
- 重新打开小程序后头像仍然保留
- 支持多个教练分别设置头像

## 🔄 头像更新逻辑
- **首页**: 实时显示最新上传的教练头像
- **详情页**: 显示对应教练的自定义头像
- **聊天页**: 使用最新的教练头像

## 📱 上传流程

### 用户头像上传
```
点击📷 → 选择拍照/相册 → 选择图片 → 上传中... → 上传成功
```

### 教练头像上传
```
点击📷 → 选择教练头像 → 选择教练 → 上传头像 → 选择图片 → 上传成功
```

## ⚠️ 注意事项
- 图片会自动压缩以提高加载速度
- 大图片上传可能需要等待时间
- 上传失败时会显示错误提示
- 可以随时更换头像，新的头像会立即生效

- ✅ 教练详情页面
